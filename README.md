# 群聊总结 Group Chat Summarizer

> 智能群聊总结，把混乱的对话变成清晰的行动指南。

## ✨ 特色

- 🇨🇳 **中文优先**：为中国用户优化的界面和文档
- 🎯 **30+ 触发方式**：自然语言即可触发，无需记忆命令
- 📊 **6 大输出模块**：基本信息、脉络梳理、核心要点、待办事项、跟进建议、其他备注
- 🔧 **8 平台支持**：飞书、钉钉、企业微信、Discord、Slack、Teams、Telegram
- 🚀 **3 级总结模式**：简要版、标准版、详细版

## 🚀 快速开始

### 一句话触发

```
"总结一下群聊"
"整理聊天记录"
"今天群里说了什么"
"提取待办事项"
"分析群聊氛围"
```

### 安装

```bash
# 通过 ClawHub 安装
clawhub install group-chat-summarizer-cn

# 或手动下载
wget https://github.com/zhuqingsonga/group-chat-summarizer-cn/releases/download/v1.0.0/group-chat-summarizer-cn.skill
```
